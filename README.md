# City Library Management System

A Java console application to manage a city library's book inventory, member records, and lending operations. The system uses Java Collections and basic object-oriented principles to provide simple book issuing and returning functionality.

## Features

- Add new books and members with unique IDs.
- Issue and return books with status updates.
- Display details for all books and members.
- Internal management with Java Maps and Lists.
- Console outputs to demonstrate functionality.

## Classes Overview

- Book: Represents a book with ID, title, author, category, and issue status.
- Member: Represents a member with ID, name, email, and a list of borrowed books.
- Library: Manages the collections of books and members, handles lending and returning books, and displays information.
- Main class: Contains the main method demonstrating basic library operations.


### Running the Program

1. Clone or download the repository.
2. Compile the project files:
javac Main.java
3. Run the program:
java Main


### Usage

- Adds sample books and members.
- Demonstrates issuing and returning of books.
- Lists all books and members on the console.
